Retrieve one captured Chimera oracle result by id.

Use this tool after `chimera_oracle_recent` or when a specific oracle id is known and you need the raw captured shell or LSP evidence plus the mutation provenance linked to that result.

The runtime captures raw shell command results and raw LSP diagnostics automatically. This tool returns recorded evidence only; it does not rerun commands, parse test output, classify shell command categories, or create obligations.

Parameter rules:
- `oracleID` is required and must identify an existing oracle result in the current project store.
- `maxOutputChars` limits shell output included in the structured result; defaults to 20000 and is capped at 200000.

Output shape:
- The result contains structured `oracle`, `linkWindow`, and `linkedChanges` data.
- `oracle.payload.shell` contains the raw captured shell command, cwd, exit, output, truncation state, and output path when available.
- `oracle.payload.lsp` contains the raw captured LSP diagnostics and diagnostic count.
